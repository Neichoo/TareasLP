Para lograr una exitosa ejecución del programa, se debe descargar SWI-Prolog, luego se puede
consultar cada archivo mediante esa aplicación. Hay 2 formas de hacerlo:
- Arriba a la izquierda del programa, donde dice "File", se clickea y luego se pone "Consult...", después se
selecciona el archivo y se comienzan a ingresar consultas.
- Se escribe en la consola de SWI-Prolog "consult('nombreArchivo.pl'). (En donde nombreArchivo.pl se refiere
a p1.pl, p2.pl, etc.)
