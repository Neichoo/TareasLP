Para lograr una exitosa ejecución del programa, en la misma carpeta donde esté ubicado el programa se necesita del archivo 'problemas.txt' y
a partir de este, se generará el archivo de salida: 'desarrollos.txt'.

Nota importante:
El programa puede tener un fallo al verificar un caso borde en específico, cuando se genera una expresión de la forma: "numero // ANS", en este caso,
el programa no puede identificar el error ya que es un problema que surge sobre la marcha y tiene conflictos en como está construido el programa, aunque
se mostrará igualmente un mensaje en la consola indicando el problema.
El mensaje proporcionará una línea aproximada donde se encuentra el error, pero puede tener un margen de error de +-1 línea.
