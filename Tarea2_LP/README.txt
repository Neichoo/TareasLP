Para lograr una exitosa ejecución del programa, en la terminal, se deben tener en cuenta los 
siguientes comandos:
"make TreasureFinder" : Este comando compilará el programa en su totalidad.
"make run" o "make runvalgrind": Este comando ejecuta el programa para que se pueda utilizar, la primera
opción es una ejecución normal y la segunda es para comprobar que no haya leaks de memoria.
"make remove": Este comando eliminará el programa que se compiló de su computadora para que no ocupe espacio.

Una vez inicializado el programa, el usuario simplemente debe ingresar por consola los inputs que se le indican en pantalla.
Por lo dicho en el foro, se asume que el usuario solo ingresará inputs correctos.

Nota:
En el juego la bomba que es de tipo ExplosionPunto nunca se logrará a visualizar en pantalla mediante MostrarTablero ya que como
explota luego de un turno, no da el espacio para ser visualizada.
